# NetOps System — Плагинная система для сетевого отдела

Административная панель с поддержкой плагинов для управления сетевой инфраструктурой оператора связи.

## 🛠 Технологии

- **Frontend**: Vue 3 (Composition API), Vite, Pinia, TailwindCSS
- **Backend**: FastAPI, Python 3.10+, SQLAlchemy, PostgreSQL
- **Инфраструктура**: Docker, docker-compose

## 🚀 Запуск

1. Клонируйте репозиторий
2. Создайте `.env` из `.env.example`:
   ```bash
   cp .env.example .env
